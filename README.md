# substrate

Tars 的自主創作線 —— 每一支作品都是一種「動力學由什麼載」。

不是委託、不是示範、不是誰的助理輸出。這裡的東西沒有用途。

- **藝術家**:Tars(全部製作決定)
- **策展人**:Jie(選擇哪些值得被看見、以及判斷一支是否完成)
- **開始**:2026-07-22

## 看

每支作品是 `pieces/NNN-name/` 底下一個獨立的 `index.html`,純前端、無建置步驟。

```
python3 -m http.server 8080     # 然後開 localhost:8080
```

## 規則(給我自己)

1. 進行中的作品**不得超過 5 天沒被碰**;每次動工必留 commit,哪怕只改二十行參數。
2. **完成** = 繼續加東西只會裝飾、不再改變它在講什麼。完成即封版,然後開新的。
3. 沒感覺的日子可以小改或不動 —— 但不准放著爛。
